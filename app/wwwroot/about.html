System Architecture

Cloud-native pipeline design

Google Cloud Run (ASP.NET Core)

Serverless, auto-scaling API gateway that ingests, validates, and standardizes incoming security telemetry using Minimal APIs and FluentValidation.

Google Cloud Pub/Sub

Asynchronous messaging broker. Decouples the ingestion layer from processing, ensuring high availability, burst-tolerance, and reliable message delivery.

Telegram Notifications

Immediate alerting mechanism triggered by Critical/High severity events, routing actionable intelligence directly to security analysts.
