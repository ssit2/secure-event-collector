Project Specifications

Enterprise Features
  • Zero-Trust Architecture: Workload Identity Federation (WIF) keyless service authentication configured via GitHub Actions.
  • Advanced Observability: Request-ID and Correlation-ID tracking middleware injected on log contexts for tracing.
  • Alpine Distroless Container: Run as non-root user (`app`) with read-only execution directories.
Infrastructure Stack
  • GCP Cloud Run: Scale-to-zero serverless orchestrator with health probes and resource limits.
  • Terraform IaC: Standard declaration of Artifact Registry, least-privilege Service Accounts, and WIF providers.
  • FluentValidation & Swagger: Dynamic verification and schema documentation with structured JSON exception logs (RFC 7807).
